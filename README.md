# Fitness AI Webapp

A fitness tracking and AI workout recommender built with **Next.js 16+** and **Tailwind CSS 4+**.  
It adapts workouts based on progress, weather, and user goals, with support for indoor/outdoor sessions, strength, cardio, mobility, and weight tracking.
